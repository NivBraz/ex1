welcome to cancel order movie

for showing all the order movies (http verb:get) please enter:
https://peaceful-waters-26893.herokuapp.com/showordermovies
in return you get a JSON that look like this:
orderd movies: [
{
id: 16,
name: "Kill bill",
category: "action",
director: "Quentin Tarantino",
language: "english",
status: "order",
actors: [
"Uma Thurman",
"David Carradine",
"Daryl Hannah"
]
}]

for looking a specific movie by id (http verb:post) enter
https://peaceful-waters-26893.herokuapp.com/showMovieById/
and send in body under 'movieId' the require id number.
in return you get the same JSON as before and if there is'nt any movie with this id you get an error JSON that look like this :
{error : "movie not found"}

for searching a movie by his language and status (http verb:get) enter:
https://peaceful-waters-26893.herokuapp.com/getstatuslangmovie/status/langauge
status can be (order,watched), language can be any language you want (english, italian,...).
in return you get the same JSON and error JSON as before, if there are multiple movies with those details the return will be array of JSON.
